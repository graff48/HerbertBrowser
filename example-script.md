# Example Browser Script

This is a sample instruction file for HerbertBrowser.

## Instructions

> Navigate to example.com first
- go to https://example.com

> Wait for the page to load
- wait 2

> Click the "Learn more" link
- click Learn more (wait: 3)

> Scroll down to see more content
- scroll down

> Navigate to another page
- go to https://httpbin.org/forms/post

> Wait for form to load
- wait 2

> Fill out the form
- type John in custname
- type 555-1234 in custtel
- type john@example.com in custemail

> Select a pizza size (radio button)
- click Medium

> Select toppings
- click bacon
- click cheese

> Submit the form
- submit form
